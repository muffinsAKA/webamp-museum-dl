# webamp-museum-dl

downloads all existing skins from the Webamp Museum @ https://skins.webamp.org/

will update from where you left off last time if you want to get the latest skins. lord willing.


# Requirements

tqdm
```
pip install tqdm
```

requests
```
pip install requests
```
